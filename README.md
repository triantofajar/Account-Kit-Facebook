## Step by Step
1. Create RN App
2. Downgrade ver. RN + hapus node_modules and package-lock.json
3. npm install
4. Install react-native-fbsdk
5. Link react-native-fbsdk
6. Setup Java Android
7. Create App in developer.facebook.com
8. Setup Java Android (App ID)
9. Setup RN App
